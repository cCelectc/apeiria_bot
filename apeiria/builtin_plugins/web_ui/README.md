# Web 管理面板

_✨ Apeiria 受保护的 Web UI 控制面板 ✨_

</div>

## 📖 介绍

这是 Apeiria 的内置管理插件，也是受保护的 Web UI 控制面板入口。

适合这些场景：

- 查看运行状态、日志与最近事件
- 管理插件、驱动、适配器与配置
- 管理访问控制、群开关与群插件状态
- 浏览数据表
- 管理自己的 WebUI 账号与会话

## 💿 安装

<details open>
<summary>作为 Apeiria 内置插件使用</summary>

Web 管理面板为 Apeiria 内置插件，默认随框架加载，无需单独安装。

如果你关闭了此插件，可在项目插件配置中重新启用：

    apeiria.builtin_plugins.web_ui

</details>

## ⚙️ 配置

在 Apeiria 的插件配置中可调整以下配置项：

| 配置项 | 必填 | 默认值 | 说明 |
|:---:|:---:|:---:|:---:|
| `token_expire_days` | 否 | `7` | WebUI 登录令牌有效天数 |

## 🎉 使用

### 打开方式

启动后访问：

    http://127.0.0.1:8080/

实际地址以启动日志为准。

## 注意

- 日常管理建议优先使用 WebUI。
- 账号恢复、注册码管理等宿主机级操作建议通过 CLI 完成。
- 涉及插件启停、配置修改、安装卸载时，注意面板中的重启提示。
