# 主人管理

_✨ Apeiria 面向主人的聊天管理命令 ✨_

</div>

## 📖 介绍

这是 Apeiria 的内置管理插件，用于让机器人主人直接在聊天环境中完成管理操作。

适合这些场景：

- 查看运行状态、驱动、适配器与当前会话信息
- 管理当前群的 Bot 开关、插件启停与访问状态
- 查看插件总览、插件详情与配置摘要
- 调整访问控制规则
- 查看、暂停、恢复定时任务
- 安排 Bot 重启

## 💿 安装

<details open>
<summary>作为 Apeiria 内置插件使用</summary>

主人管理为 Apeiria 内置插件，默认随框架加载，无需单独安装。

如果你关闭了此插件，可在项目插件配置中重新启用：

    apeiria.builtin_plugins.admin

</details>

## ⚙️ 配置

该插件目前无独立配置项。

## 🎉 使用

### 指令表

| 指令 | 权限 | 需要@ | 范围 | 说明 |
|:---:|:---:|:---:|:---:|:---:|
| `/admin` | 主人 | 否 | 群聊 / 私聊 | 查看管理总览 |
| `/status` | 主人 | 否 | 群聊 / 私聊 | 查看运行状态摘要 |
| `/sid` | 主人 | 否 | 群聊 / 私聊 | 查看当前会话 SID 与 UID |
| `/adapters` | 主人 | 否 | 群聊 / 私聊 | 查看已加载适配器 |
| `/drivers` | 主人 | 否 | 群聊 / 私聊 | 查看驱动信息 |
| `/plugins` | 主人 | 否 | 群聊 / 私聊 | 查看插件总览 |
| `/plugin info <插件名>` | 主人 | 否 | 群聊 / 私聊 | 查看插件详情 |
| `/plugin enable\|disable <插件名>` | 主人 | 否 | 群聊 / 私聊 | 启用或禁用插件 |
| `/plugin configs <插件名>` | 主人 | 否 | 群聊 / 私聊 | 查看插件配置摘要 |
| `/config core` | 主人 | 否 | 群聊 / 私聊 | 查看核心配置摘要 |
| `/config plugin <插件名>` | 主人 | 否 | 群聊 / 私聊 | 查看插件配置摘要 |
| `/access ...` | 主人 | 否 | 群聊 / 私聊 | 管理访问规则 |
| `/tasks` | 主人 | 否 | 群聊 / 私聊 | 查看调度任务列表 |
| `/task info\|pause\|resume <任务ID>` | 主人 | 否 | 群聊 / 私聊 | 管理指定任务 |
| `/restart` | 主人 | 否 | 群聊 / 私聊 | 安排 Bot 重启 |

## 注意

- 这是高权限插件，只适合机器人主人使用。
- 涉及插件启停、访问规则或配置变更时，部分效果需要重启后完全生效。
- 日常图形化管理建议优先使用 `apeiria.builtin_plugins.web_ui`。
