# 主人管理

_Apeiria 面向主人的聊天管理命令_

</div>

## 介绍

Apeiria 内置管理插件，用于让机器人主人直接在聊天环境中完成系统管理操作。

适合这些场景：

- 查看运行状态与当前会话信息
- 管理插件启停与访问控制
- 查看插件详情与配置摘要
- 调整访问控制规则
- 安排 Bot 重启（支持重启后自动通知）

> AI 相关管理命令（`/persona`、`/model`、`/ai`、`/rel`）由 AI 插件提供，不在本插件范围内。

## 安装

<details open>
<summary>作为 Apeiria 内置插件使用</summary>

主人管理为 Apeiria 内置插件，默认随框架加载，无需单独安装。

如果你关闭了此插件，可在项目插件配置中重新启用：

    apeiria.builtin_plugins.admin

</details>

## 配置

该插件目前无独立配置项。

## 使用

### 指令表

| 指令 | 权限 | 范围 | 说明 |
|:---|:---:|:---:|:---|
| `/status` | 主人 | 群聊 / 私聊 | 查看运行状态（运行时长、插件数、适配器） |
| `/sid` | 主人 | 群聊 / 私聊 | 查看当前会话的 Bot ID、User ID、Session ID |
| `/plugins` | 主人 | 群聊 / 私聊 | 查看插件总览（启用/禁用/保护状态） |
| `/plugin info <插件名>` | 主人 | 群聊 / 私聊 | 查看插件详情（版本、依赖、配置） |
| `/plugin enable <插件名>` | 主人 | 群聊 / 私聊 | 启用插件 |
| `/plugin disable <插件名>` | 主人 | 群聊 / 私聊 | 禁用插件（受保护插件会拒绝） |
| `/plugin configs <插件名>` | 主人 | 群聊 / 私聊 | 查看插件配置摘要 |
| `/config core` | 主人 | 群聊 / 私聊 | 查看核心配置摘要 |
| `/config plugin <插件名>` | 主人 | 群聊 / 私聊 | 查看指定插件的配置摘要 |
| `/access plugin <插件名>` | 主人 | 群聊 / 私聊 | 查看插件的访问规则汇总 |
| `/access rule <allow\|deny> <user\|group> <ID> <插件名>` | 主人 | 群聊 / 私聊 | 添加或更新访问规则 |
| `/access remove <user\|group> <ID> <插件名>` | 主人 | 群聊 / 私聊 | 删除访问规则 |
| `/restart` | 主人 | 群聊 / 私聊 | 重启 Bot（完成后自动回报耗时） |

### 重启行为

`/restart` 会在重启完成后向触发命令的会话发送通知（包含耗时）。支持跨平台适配器：根据触发时的平台类型选择对应的消息发送方式。如果通知发送失败，会重试最多 3 次。

## 注意

- 这是高权限插件，只适合机器人主人使用。
- 插件启停、访问规则变更后部分效果需要重启才能完全生效。
- 日常图形化管理建议优先使用 Web UI。
